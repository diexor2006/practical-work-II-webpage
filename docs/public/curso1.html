Year 1

Building a thorough foundation across several areas that are necessary for contemporary engineering and technology is the main goal of this academic year. Students learn about software development in Fundamentals of Computer Engineering, which emphasizes programming and cooperative teamwork. Topics covered include the software life cycle, version control systems like Git, and the usage of development tools like Visual Studio Code. Financial accounting gives students the fundamentals of financial management, including creating balance sheets and evaluating financial statements, to improve decision-making, while logic and critical thinking hone analytical reasoning and problem-solving abilities by emphasizing logical structures and avoiding biases. By introducing the fundamentals of data manipulation, visualization, and large-scale data management with frameworks like Hadoop, the Foundations of Data Science and Big Data course equips students to extract insights from intricate datasets. Students study circuits, microprocessors, embedded systems, and the interaction between hardware and software in Electronics and Computer Technology. In order to improve corporate processes and maximize decision-making through data-driven strategies, the Technological Tools for corporate and Data Analytics course examines tools such as Tableau and Power BI. In order to describe and solve engineering problems, Maths for Engineering II offers sophisticated mathematical techniques such as differential equations, calculus, and linear algebra. Last but not least, Introduction to Web Programming equips students with the knowledge of client-server architecture and the skills necessary to create dynamic web applications using HTML, CSS, and JavaScript. This allows them to design, develop, and implement interactive web solutions. Together, these courses seek to foster the critical technical knowledge, critical thinking, and practical abilities required to tackle real-world engineering, technological, and business challenges. intricate data sets.
